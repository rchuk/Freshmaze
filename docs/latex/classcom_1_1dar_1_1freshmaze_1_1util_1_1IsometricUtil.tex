Converts vector from isometric to cartesian coordinate system.

The original formula is the following\+: 
\begin{DoxyPre}
{\ttfamily \newlinereturn new Vector2(\newline     0.5f * (2.0f * vec.y + vec.x),\newline     0.5f * (2.0f * vec.y -\/ vec.x)\newline);\newline}
\end{DoxyPre}

\begin{DoxyPre}
{\ttfamily \newline return new Vector2(\newline     0.5f * (-\/2.0f * vec.y + vec.x),\newline     0.5f * (2.0f * vec.y + vec.x)\newline );\newline }
